## 使用说明 & 备注

- **快速跳脱注释**  
  在插入模式下，按 `Ctrl-u` 可删除当前行自动注释前缀（如果存在）。

- **常用命令**  
  `gg` —— 快速跳转到文件开头。

- **Markdown 插入表格**  
  快速插入 Markdown 格式的表格结构。

- **查看定义'g+d'，返回'crlt+o'**
